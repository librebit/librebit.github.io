[Errores Linux Mint] Ciclo en Inicio De Sesión

Mar 26, 2018 • Iván Rodríguez

El Display Manager de Linux Mint mdm está asociado con el inicio de sesión de Linux Mint tanto para cinnamon como para mate, un síntoma de que se ha dañado el programa mdm es que al iniciar sesión no se pueda entrar correctamente a la sesión y aparezca un error relacionado a .Xauthority indicando también que se volverá a iniciar sesión después de unos cuantos segundos quedando el inicio de sesión en un ciclo.

Este error se presenta normalmente después de actualizar el sistema operativo a través de apt upgrade o por el Update Manager y con programas relacionados al sistema de arranque UEFI.

Solución.

Para corregir el problema de inicio de sesión ciclado se debe de acceder a la terminal de comandos, una forma de ingresar es presionando

ctr + alt + F2

Esto abrirá una terminal de comandos en donde se podrá inciar sesión con el usuario de la computadora, una vez iniciada la sesión se debe de ejercutar los comandos de actualización del sistema operativo.

sudo apt update
sudo apt upgrade

Cabe mencionar que aunque en esa sesión no se tenga el modo gráfico activado, el sistema operativo se conectará a las redes alámbricas o inalámbricas las cuales se hayan configurado previamente, por lo que el acceso a los repositorios no se verá afectado.

Una vez actualizado los repositorios se tiene que desinstalar e instalar el Display Manager de Linux Mint, los comandos para realizar esta acción son

sudo apt purge mdm
sudo apt install mdm

En el proceso de instalación preguntará por el display manager por defecto de Linux Mint, las opciones son gdm que es el de Gnome y mdm que es el utilizado por Linux Mint. En caso de escoger o querer cambiar a Gnome se debe de instalar gdm primeramente.

sudo apt install gdm

Una vez instalado el Display Manager por defecto se procede a reiniciar el sistema operativo

sudo reboot now

En caso de que se haya configurado el inicio de sesión automático este se verá desactivado, para activarlo es necesario habilitar la opción dentro de Login Window en las preferencias del sistema operativo una vez iniciada la sesión en modo gráfico.
